# 📊 Sales Performance Dashboard using Power BI

## 📌 Project Overview

This project presents an interactive Sales Performance Dashboard built using Microsoft Power BI. The dashboard provides insights into sales, cost, profit, order trends, and regional performance through interactive visualizations and filters.

The objective is to transform raw sales data into meaningful business insights that support better decision-making.

---

## 🚀 Features

- Interactive Power BI Dashboard
- KPI Cards for:
  - Total Sales
  - Total Cost
  - Total Profit
  - Total Orders
- Monthly Sales Trend Analysis
- Sales by Territory
- Sales by Quantity
- Profit by Territory
- Sales vs Cost Relationship (Scatter Plot)
- Interactive Filters:
  - Month
  - Territory
  - Quantity
- Actionable Business Insights

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📂 Dataset

The dataset contains sales transaction records with the following fields:

- SalesOrderNumber
- OrderDate
- ProductKey
- ResellerKey
- EmployeeKey
- SalesTerritoryKey
- Quantity
- Unit Price
- Sales
- Cost

## 📊 Key Insights

- Territory 4 generated the highest sales revenue.
- Sales peaked during August and November.
- Quantities of 2–3 units contributed the largest share of revenue.
- Sales and cost show a strong positive relationship.
- Profit margins are relatively low compared to total sales, indicating opportunities for cost optimization.
